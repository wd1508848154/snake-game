# 🐍 贪吃蛇游戏

一个使用纯 HTML + CSS + JavaScript 制作的贪吃蛇网页游戏，支持桌面和移动端。

## 🎮 在线试玩

👉 **[点击开始游戏](https://你的用户名.github.io/snake-game/)**

## ✨ 特性

- 🖥️ **桌面端**：使用方向键控制
- 📱 **移动端**：触屏方向键控制
- ⚡ 速度逐渐加快，挑战你的反应
- 💫 霓虹发光视觉效果
- 🎯 实时得分显示

## 🕹️ 操作方式

| 平台 | 操作 |
|------|------|
| 电脑 | 方向键 ↑↓←→ 移动，空格键暂停 |
| 手机 | 点击屏幕下方的方向按钮 |

## 🚀 本地运行

直接在浏览器中打开 `index.html` 文件即可游玩。

## 📝 技术栈

- HTML5 Canvas
- Vanilla JavaScript
- CSS3 (渐变、阴影、动画)
